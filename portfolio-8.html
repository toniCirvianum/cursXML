Web Responsive

Per fer la web responsive s'ha creat una fulla d'estils CSS que es carrega quan la pantalla fa menys de 768px.

El que fa la fulla d'estils es convertir els menús, de horitzontal passa a vertical i també fa el mateix amb la disposición dels quadrats.

Disseny del portfoli

Per al disseny del portfoli s'ha partit d'una plantilla amb un disseny de quadricula per afegir les diferents activitats del portfoli. Al clicar a cada quadre s'accedeix a una plantilla on es mostra informació en format de dues columnes.

RGPD

Quan es carrega la web, apareix el banner de la RGPD amb un botó per acceptar les cookies.

Quan es clica a l'enllaç senyalat amb la fletxa de color vermell t'envia a la web per poder llegir els diferents textos del RGPD.
