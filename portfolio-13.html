Orla de Minerals

Al hosting s'ha creat una base de dades de minerals i mitjançant el següent codi en php, es consulta la informació dels minerals i es visualitza en forma d'ORLA.

Fent clic al botó es pot visualitzar l'orla. Per visualitzar una orla amb un grup determinat d'animals, només cal afeir a la url el text ?grup=nom del grup

Orla d'equips de l'ACB

A partir d'una base de dades dels equips de l'ACB, en format excel, s'ha transformat en un fitxer sql mitjançant la web de RebaseData. Després seguint l'exemple dels minerals, s'ha creat un codi en php per visualitzar l'orla d'un equip de l'ACB.

Igual que en l'exemple anterior, afegint el text ?Equip=nom_del_equip es pot visualitzar l'orla de qualsevol equip de l'ACB.
